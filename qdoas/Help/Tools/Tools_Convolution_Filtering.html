QDOAS Convolution/Filtering Tool : Filtering page

After convolution, a low-pass and/or a high-pass filtering can be applied on the cross-section.  By default, no filtering is applied. The presentation of this page is the same as the Filtering one of Projects Properties.  According to the selected filter type, the requested information can be different. 

The original spectrum can be divided by the smoothed one or the smoothed spectrum can be subtracted from the original one.  The second feature is useful to create differential cross-sections.
